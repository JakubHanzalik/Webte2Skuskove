Application Guide

Login and Registration

Users can register and log in to access their profiles. The registration component handles user sign-ups while the login component manages user authentication.

Dashboard

The dashboard is the main interface where users can submit questions, scan QR codes, or view their activity. It's responsive, ensuring usability across various devices.

Question Handling

Questions can be of three types: Poll, Numeric, and Alphanumeric. Users submit questions, which are then dynamically generated and displayed in the question component.

Admin Panel

The admin panel allows administrators to manage users, questions, and view system analytics. Admins can activate/deactivate questions, change user roles, and export data.

Exporting Data

Both users and admins can export data. Users can export their responses, while admins can export comprehensive user and question data.
